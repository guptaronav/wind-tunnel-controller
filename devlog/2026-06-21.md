# Devlog #2 — V1 Complete
**June 21, 2026**

V1 is done. All parts printed, assembled, and tested — the tunnel runs end to end with the mist maker producing visible flow and the knob controller driving the fan. It works. That said, testing surfaced a few clear improvements for V2: the diffuser fan needs screw mounts (it's not secure enough right now), the test section could use a visual overhaul to look more intentional, and all the electronics need a proper enclosure rather than sitting loose. V2 scope is locked — structural fix for the diffuser, cosmetic pass on the test section, and a housing for the controller and wiring.
